003 Control de Clase Hello World Kotlin

Este proyecto es una aplicación web simple (API REST) desarrollada con Kotlin y Spring Boot para demostrar la creación de variables, clases y el uso de anotaciones de controlador.

Cómo Ejecutarlo
Ejecutar la aplicación: Localiza el archivo Example.kt (el que contenga la función main) y haz clic en el botón verde de "Run" (Reproducir) en la barra superior de IntelliJ.
Verificar el Estado: Una vez que la aplicación esté ejecutándose (http://localhost:8080), podrás acceder a los endpoints.
Endpoints Disponibles
Puedes acceder a las siguientes URLs en tu navegador web o usando una herramienta como Postman:

Para obtener el mensaje "Hola Mundo!": http://localhost:8080/hello

Para obtener el objeto JSON del cuadrado: http://localhost:8080/square
